# Rabbiya Portfolio — React + Vite

A ready‑to‑run portfolio starter with:
- Animated particle background (tsparticles)
- Animate‑on‑scroll (AOS)
- Framer Motion hero
- Tailwind CSS
- Glassmorphism components
- Responsive sections similar to your reference

## Quickstart
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
